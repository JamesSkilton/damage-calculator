# Phase 1 Foundation: Vite + React + TypeScript Startup

This repository now has a Vite-powered React + TypeScript foundation for the rewrite.

## Scripts

- `npm run dev` — start local development server
- `npm run build` — type-check + production build
- `npm run preview` — preview production build locally
- `npm run typecheck` — TypeScript project type-check

## Runtime config

- `VITE_APP_TITLE` — optional app title override for the shell and document title
- `VITE_DEFAULT_MODE_SLUG` — optional default route slug; must match a configured calculator mode

## Getting started

1. Install dependencies:
   - `npm install`
2. Start development:
   - `npm run dev`
3. Build for production:
   - `npm run build`
